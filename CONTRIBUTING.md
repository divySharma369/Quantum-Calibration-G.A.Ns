# Contributing to Quantum-GAN-Calibration

We welcome contributions! Whether you're fixing bugs, improving documentation, or adding new quantum noise models, your help is appreciated.

## How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Branch**: `git checkout -b feature/amazing-feature`.
3. **Commit Your Changes**: Use descriptive commit messages.
4. **Push to Your Branch**: `git push origin feature/amazing-feature`.
5. **Open a Pull Request**: Explain your changes and wait for review.

## Coding Standards

- Follow PEP 8 for Python code.
- Ensure all tests pass (`pytest`).
- Document new functions and classes.
